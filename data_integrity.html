Backup and Restore

Full Backup

RewardLedger allows you to download a full backup of your data.

  • Includes all user data and configuration
  • Stored as a JSON file
  • Recommended before major changes

Restore

Restoring a backup will replace your current data.

  • This action cannot be undone
  • Always backup before restoring
  • Requires typing RESTORE to confirm

Wipe Ledger Entries

This tool clears all transaction data while preserving your setup.

  • Useful for testing or resetting tracking
  • Requires typing WIPE
