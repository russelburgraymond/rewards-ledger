Settings

Settings centralizes app management pages so setup data, AI Import defaults, and other utilities stay organized in one place.

What lives under Settings
  • Assets
  • Accounts
  • Categories
  • Apps
  • App order can be changed by drag and drop on the Apps tab.
  • Miners
  • Referrals
  • Accounts, Miners, and Referrals can now also be reordered by drag and drop.
  • Dashboard Settings
  • AI Import
  • Other
  • System Status
Dashboard Settings

Dashboard Settings controls which category tiles appear on the dashboard and how those tile totals behave.

  • Dashboard date range switches dashboard tiles between All Time totals and Current Year totals.
  • Number of dashboard tile rows controls how many drop rows are shown by the dashboard layout editor.
  • Show Value at Receipt / Cost at Payment totals adds those rollups to eligible dashboard tiles.
Calculated Tiles in Dashboard Settings

The Dashboard settings tab can now create Calculated Tiles.

  • Give the tile a name such as Approximate Gross.
  • Add one or more category lines.
  • Choose whether each line should Add or Subtract.
  • Choose whether each line uses Absolute Amount or Raw Signed Amount.

This allows you to build custom dashboard math without hard-coding new dashboard tiles into the app.

AI Import settings

The AI Import settings page now has a clearer split between Rewards Screen and Wallet.

  • Preferred Tracking Mode chooses which screenshot type is selected by default on the AI Import page.
  • Default Categories by Screenshot Type controls which category boxes start checked for Rewards Screen imports and Wallet imports.
  • Category Aliases are grouped by screenshot type so you can keep Wallet wording separate from Rewards Screen wording.
  • Asset Aliases stay shared because BTC, GMT, USD, and similar labels are reused across both screenshot models.
Why AI Import aliases are grouped now

Rewards Screen and Wallet do not label the same information the same way. Grouping aliases by screenshot type helps avoid confusion and gives cleaner control over what AI Import is allowed to look for in each mode.

Apps tab

The Apps tab lets you add, edit, activate, and reorder apps. Drag rows in the app list using the handle on the left to update display order quickly.

  • New apps are now added to the end of the list automatically using the next available sort order.
  • This prevents multiple newly added apps from all starting at sort order 0.
Accounts, Miners, and Referrals ordering

The Accounts, Miners, and Referrals tabs now support drag-and-drop row reordering just like Apps and Templates.

  • Use the handle on the left side of each row to drag items into a new order.
  • New accounts, miners, and referrals are added to the end of the list automatically using the next available sort order.
  • This prevents multiple new rows from being created with a sort order of 0.
Other tab

The Other tab is where backup and restore tools belong, along with other app-wide utility actions that do not fit a normal data table.

System Status

Use System Status to verify important technical details about the running app and environment.

BTC Sats entry: Quick Entry can show a Sats checkbox when BTC is selected. Saved Quick Entry items can default this on so users can enter values like 333 instead of 0.00000333.
