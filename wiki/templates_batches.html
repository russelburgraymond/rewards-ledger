Templates & Batch-Style Entry

Templates let you save reusable entry layouts. Using a template creates a consolidated form that can submit multiple related lines together.

What a template saves
  • Apps
  • Categories
  • Assets
  • Accounts
  • Optional field visibility settings
  • Optional default values
Using a template
BTC Sats option: Template lines can now store a default Sats preference. When a BTC line uses that option, the entry form opens with the sats checkbox ready so users can type whole sats instead of decimal BTC.
Per-line date option: Template lines can now expose their own Date field. When enabled, the line starts with the main template date filled in, but you can change that row to a different day before saving. When disabled, that line keeps using the main template date automatically.

When you click Use on a template, RewardLedger opens a single form containing the template lines. The date usually defaults to the current date, and any stored defaults can prefill the fields.

Multi-add inside templates

Some template lines support one amount per line in a textarea. This is useful when you want repeated entries using the same setup.

Date warning for batch-generated entries

If several entries were generated together from the same batch-style action, changing the date on one line may update the batch date for the related entries as well. Pay attention to any warning shown on the form.
